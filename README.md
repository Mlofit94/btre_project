This project is a full-stack real estate web application built with Django and PostgreSQL, designed to manage property listings, realtors, user inquiries, and site content in a structured and scalable way.

The application follows Django best practices, including modular apps, database migrations, and environment isolation using virtual environments. It integrates PostgreSQL for production-grade data persistence and uses Djangoâ€™s ORM to manage models and relationships efficiently.

Tech Stack:

Backend: Django

Database: PostgreSQL

Frontend: Django Templates, HTML/CSS

Image Handling: Pillow

Database Driver: psycopg2 / psycopg2-binary

Environment: Python virtual environments (venv)

Key Features:

Property listings with images and detailed metadata

Realtor management and association with listings

User contact and inquiry handling

Admin dashboard for content and data management

PostgreSQL-backed database with migrations

Static and media file handling

Production-ready project structure
